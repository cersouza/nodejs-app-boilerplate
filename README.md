# Customers API
_A NodeJS API boilerplate using Clean Architecture._

## 1. About :orange_book:

This is a NodeJS API made as personal boilerplate of project with these technologies:
- NodeJS;
- Express JS;
- Typescript;
- Jest;

## 2. Setup :computer:

```bash
npm i
```

## 3. Run in development mode :arrow_forward:

```bash
npm run dev

```

## 4. Run tests :test_tube:

```bash
npm run test

```
